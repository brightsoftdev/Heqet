================
Heqet
================

Description
---------------
Heqet is an useful utility collection for game development.
It extends cocos2d.

Installation
---------------

::

    cd your/cocos2d/project
    git submodule add git@github.com:giginet/Heqet.git project/libs/
    git submodule init
    git submodule update
